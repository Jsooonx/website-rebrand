# Asset Direction

Simpan asset visual website di folder ini ketika implementasi dimulai.

- \`images/\` untuk foto, mockup, dan tekstur.
- \`icons/\` untuk SVG icon.
- \`logos/\` untuk wordmark dan brand mark.

Gunakan nama file deskriptif, optimalkan gambar untuk web, dan sediakan
alternatif teks saat dipakai dalam interface.
