# Radian Works

Radian Works is a fictional embedded AI product studio created as a distinct
website-rebrand exploration.

The approved starting point is the full brand direction in
[BRANDING.md](BRANDING.md). The implemented site uses Vite, React, Framer
Motion, and Lenis.

## Source map

- `src/App.tsx` — page composition and section-level interaction state.
- `src/hooks/useLenisScroll.ts` — smooth-scroll lifecycle and initial anchor
  restoration.
- `src/hooks/useFloatingNavigation.ts` — inline-to-floating navigation state.
- `src/components/BrandMark.tsx` — shared folded-pathway brand asset.
- `src/styles.css` — visual system, responsive layouts, and component styles.
