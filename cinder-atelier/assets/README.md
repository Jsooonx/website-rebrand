# Asset Direction

Store original implementation assets here once the site is built.

- `images/` — hero, case-study, and atmospheric photography.
- `motion/` — line-of-heat source clips or lightweight SVG/CSS motion assets.
- `icons/` — accessible SVG interface icons.
- `logos/` — Cinder Atelier wordmark and monogram exports.

Use descriptive filenames, export responsive image sizes, and retain the source or prompt provenance for every generated visual. Do not use the supplied hero reference as an implementation asset.
